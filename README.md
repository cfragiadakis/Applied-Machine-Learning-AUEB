# Applied Machine Learning (AUEB)

This repository contains the programming assignments completed for the **Applied Machine Learning** course at the **Department of Management Science and Technology (DMST), Athens University of Economics and Business (AUEB)**.

The assignments cover a range of machine learning topics, including exploratory data analysis, statistical modelling, natural language processing, text classification, clustering, and predictive modelling.

## Assignments

### 1st Assignment: Exploring AirBnB in Athens

[`Exploring-AirBnB-in-Athens`](./Assignment-1-Exploring-AirBnB-in-Athens)

Exploratory data analysis of AirBnB listings in Athens using the Inside AirBnB dataset. The project investigates pricing, availability, host activity, occupancy estimates, and geographical patterns through statistical analysis and interactive visualizations.

---

### 2nd Assignment: Exploring Why Civil Resistance Works

[`Exploring-Why-Civil-Resistance-Works`](./Assignment-2-Exploring-Why-Civil-Resistance-Works)

Exploratory data analysis and statistical modelling based on the NAVCO 1.2 dataset to reproduce key findings from *Why Civil Resistance Works*. The project examines the effectiveness of violent and nonviolent campaigns, investigates factors associated with campaign success, and evaluates the well-known 3.5% rule.

---

### 3rd Assignment: Roget's Thesaurus Classification

[`Roget-Thesaurus-Classification`](./Assignment-3-Roget-Thesaurus-Classification)

A natural language processing project that applies web scraping, word embeddings, clustering, and supervised learning to analyze Roget's Thesaurus. The project explores the lexical hierarchy through unsupervised learning and develops machine learning models for class and section prediction.
